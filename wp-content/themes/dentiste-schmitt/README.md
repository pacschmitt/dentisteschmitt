# Dentiste Schmitt WordPress Theme

## Description

Thème WordPress moderne et professionnel spécialement conçu pour le cabinet dentaire Schmitt. Ce thème offre un design responsive, des fonctionnalités personnalisées pour les services dentaires et une interface utilisateur intuitive.

## Fonctionnalités

- **Design Responsive** : Optimisé pour tous les appareils (desktop, tablette, mobile)
- **Types de contenu personnalisés** :
  - Services dentaires
  - Membres de l'équipe
- **Zones de widgets** : 3 zones dans le footer
- **Emplacements de menu** : Menu principal et menu footer
- **Support des images mises en avant**
- **Support du logo personnalisé**
- **Support HTML5**
- **Styles personnalisés** : Palette de couleurs professionnelle adaptée au secteur dentaire

## Installation

1. Copiez le dossier `dentiste-schmitt` dans `/wp-content/themes/`
2. Activez le thème depuis l'interface d'administration WordPress
3. Configurez les menus dans Apparence > Menus
4. Ajoutez des widgets dans Apparence > Widgets

## Personnalisation

### Couleurs

Les couleurs principales peuvent être modifiées dans `style.css` :
- `--primary-color` : Couleur principale (bleu)
- `--secondary-color` : Couleur secondaire (bleu clair)
- `--accent-color` : Couleur d'accentuation (turquoise)

### Logo

Ajoutez votre logo dans Apparence > Personnaliser > Identité du site

### Menus

Deux emplacements de menu sont disponibles :
1. **Menu Principal** : Navigation en haut du site
2. **Menu Footer** : Navigation dans le pied de page

### Widgets

Trois zones de widgets sont disponibles dans le footer pour ajouter :
- Informations de contact
- Liens utiles
- Horaires d'ouverture
- Réseaux sociaux

## Fichiers du thème

- `style.css` - Styles principaux
- `functions.php` - Fonctionnalités PHP
- `header.php` - En-tête du site
- `footer.php` - Pied de page
- `index.php` - Template principal
- `page.php` - Template pour les pages
- `single.php` - Template pour les articles
- `archive.php` - Template pour les archives
- `search.php` - Template pour les résultats de recherche
- `template-parts/content.php` - Template de contenu
- `template-parts/content-none.php` - Template pour aucun résultat
- `js/main.js` - JavaScript personnalisé
